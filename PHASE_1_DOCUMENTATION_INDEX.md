### Phase 1 Features (3 Core + 2 Bonus)
1. **Contact Tagging System** - Organize contacts with tags
2. **Advanced Search** - Multi-criteria contact filtering
3. **Bulk Operations** - Update multiple contacts at once
4. **Activity Timeline** (Bonus) - Audit trail
5. **Birthday Reminders** (Bonus) - Automated date reminders

- Spring Boot 3.5

- Liquibase for migrations
- Swagger/OpenAPI for documentation

- REST API with JSON
- Security with Spring Security
